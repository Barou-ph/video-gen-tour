HUONG DAN SU DUNG
=================

LAN DAU TIEN (chi lam 1 lan):
1. Cai Python: https://python.org/downloads
   ⚠️ Tick chon "Add Python to PATH" truoc khi cai!
2. Cai FFmpeg:
   - Tai: https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-essentials.zip
   - Giai nen → vao thu muc bin → copy duong dan
   - Them vao PATH (nho lai huong dan neu can)
3. Double-click: setup.bat
   Doi 5-10 phut de cai thu vien

HANG NGAY:
- Double-click: run.bat
- Browser tu mo tai http://localhost:8501
- Nhap API keys vao sidebar lan dau, lan sau tu load tu .env
